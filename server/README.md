# Server

Серверная часть FonlineWeb. Отвечает за хранение состояния мира, обработку действий игроков и синхронизацию клиентов через WebSocket.

## Быстрый старт

```bash
npm install
npm run dev
```

Сервер слушает порт из `PORT` (по умолчанию `3000`) и принимает WebSocket-подключения на том же порту.

### REST эндпоинты

- `GET /status` — текущее количество подключённых клиентов и их идентификаторы.

### Сообщения WebSocket

- `welcome` — отправляется сразу после подключения клиента.
- `online` — рассылается всем клиентам при каждом подключении/отключении и содержит общее количество игроков.
- `error` — сервер сообщает о некорректных сообщениях.

На данный момент сервер принимает только служебное сообщение `hello` и игнорирует остальные типы, подготавливая почву для следующих этапов реализации протокола.
